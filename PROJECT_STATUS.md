## 📋 项目概况

XRF-Go 是一个简洁高效的 Xray 安装配置工具，设计理念为"高效率，超快速，极易用"。项目继承 233boy 项目的设计理念，以多配置同时运行为核心设计，专门优化了添加、更改、查看、删除四项常用功能。

### 🎯 核心设计目标

- [x] **高效率** - <1秒添加协议配置
- [x] **超快速** - 智能默认值，零配置启动
- [x] **极易用** - 直观命令设计，协议别名支持
- [x] **多配置并发** - 充分利用 Xray `-confdir` 特性
- [x] **单二进制分发** - 所有配置模板内嵌
